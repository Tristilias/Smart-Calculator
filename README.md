# Smart Calculator 🧮

Smart Calculator adalah proyek kalkulator modular berbasis C++ yang dirancang untuk membantu berbagai perhitungan sehari-hari secara cepat dan praktis.

Project ini dibuat sebagai sarana belajar:
- Modular Programming
- Multi-file C++
- Header File (`.h`)
- Function & Switch Case
- Struktur project skala kecil-menengah

---

## ✨ Features

### 🔢 Basic Calculations
Menghitung:
- Pertambahan
- Pengurangan
- Perkalian
- Pembagian

### 📐 Geometry Calculator
Menghitung:
- Luas bangun datar
- Keliling bangun datar
- Volume bangun ruang
- Luas permukaan bangun ruang

### 🌡 Temperature Converter
Konversi suhu:
- Celsius
- Fahrenheit
- Kelvin
- Reamur

### 💸 Discount Calculator
Menghitung:
- Potongan harga
- Harga akhir setelah diskon

### ⚖ BMI & Ideal Weight Calculator
Menghitung:
- BMI (Body Mass Index)
- Berat badan ideal
